# to11 Community

This is the official Community repository for to11.ai. Think of it as the ultimate fan club and stage door for the project. 

Use this repo to:
- Report bugs / Contribute / request features
- Ask questions
- Security issues


## Report Bugs / Contribute / Request Features

We operate on an issue-driven contribution model. To keep our supply chain "none more secure" against the flood of AI-generated code, we don’t accept external pull requests. Fork PRs are closed automatically— it’s not gatekeeping, it’s just making sure the "stage" stays clear of unverified gear. Instead we follow this process: 

- You file the Issue: Bugs & Feature requests are the soul of the project. Give us the details.

- We Triage: A maintainer works with Claude to trace the codebase and craft a solid implementation plan. We don't "jazz it up" until the plan is perfect.

- We Build: to11 engineers (directing our own AI roadies) implement the fix with full test coverage and a heavy-duty review.

- You Get Credit: You’re the co-songwriter. We’ll gladly list you as a co-author on the PR.

The Result: You get the features you need, maintained by us, without the "heavy lifting" of keeping a fork in sync.


## Security Issues

If you’ve spotted a vulnerability or a security "bad smell" in our gear, keep it off the main stage. Opening a public issue for a security flaw is like shouting the band’s hotel room number into a live mic.  

We’ve enabled private vulnerability reporting for this repository to keep things tight and professional. Please use that private channel here to alert the roadies, and we’ll handle it behind the velvet rope.
